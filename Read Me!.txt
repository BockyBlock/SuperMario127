Thank you for downloading the Super Mario 127 0.6.1 demo! 
This game is still in development, so expect bugs.

Default controls:
Z - Jump
X - Spin
C - Use Fludd
Shift - Dive
V - Switch Nozzles
Left/Right - Move
Down - Ground Pound, Crouch, Butt Slide (on slopes)
Up - Ground Pound Cancel

P - Pause
M - Mute Music
3 - Toggle Photo Mode
Alt + Enter - Fullscreen

Editor:
WASD Keys - Move Editor Camera
Left Click - Place Tiles/Objects
Right Click - Delete Tiles/Objects

T - Switch Between Play and Edit
I - Switch Layer View Mode
L - Switch Layers
H - Switch Object Placement Mode
Hold CTRL - Precise Object Placement
R - Reload Current Level
R (while hovering over object) - Rotate Object
Y (while hovering over object) - Flip Object Vertically
F (while hovering over object) - Flip Object Horizontally
K (while hovering over object) - Enable/Disable Object
Shift + Left Click (while hovering over object) Duplicate Object
Mouse Wheel (while hovering over certain objects) - Change Special Property

You can access in-game options using the pause menu (P).
